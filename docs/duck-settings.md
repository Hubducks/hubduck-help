# Duck Settings

Each duck has its own configuration for access, digest, and sending permissions.

## Accessing settings

From the duck inbox, click **⚙ Duck Settings** in the top-right corner. Or go to **Duck Settings** in the sidebar.

## General settings

- **Duck Name** — How the duck appears in the sidebar and header
- **Duck Email** — The forwarding address (e.g., `staff@kilcolganetns.hubduck.net`)
- **Timezone** — Used for digest scheduling and event times
- **Error Alerts** — Get notified when email processing fails

## Creating a new duck

1. Click **+ New Duck** in Duck Settings
2. Enter a name (e.g., "3rd Class", "Staff Room", "Board")
3. The email address is auto-generated from the name
4. Choose a type:
   - **Staff Room** — All staff can see emails (whole-school communications)
   - **Class Group** — For one class or teacher (class-specific communications)
5. Click **Create Duck**

## Access settings

Control who can access the duck and how:

- **Domain Only** — Only school email addresses can be invited
- **Invite Only** — Only explicitly invited people
- **Public View** — Anyone can view, only subscribers can interact

## Who Can Send

Control who can send emails to this duck:

- **Allow all subscribers** — Everyone with access can forward emails
- **Allow selected subscribers** — Only specific people can send
- **Allowed senders** — External email addresses or domains that are whitelisted
